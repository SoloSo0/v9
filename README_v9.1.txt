Phage + Antibiotic Selector (PhageATB) v9.1
==========================================

Текущая версия: 9.1 (Нативная)
Дата обновления: 2026-04-23

ОПИСАНИЕ:
Профессиональное программное обеспечение для анализа синергии бактериофагов и антибиотиков. 
Версия 9.1 является важным обновлением, полностью заменяющим веб-интерфейс на нативное приложение Windows.

ОСНОВНЫЕ ИЗМЕНЕНИЯ В 9.1:
1. Архитектура: Полный отказ от Streamlit. Приложение теперь работает на базе CustomTkinter, что обеспечивает мгновенный запуск и отсутствие необходимости в браузере.
2. Стабильность: Исправлена критическая ошибка "TypeError: CTkFont got an unexpected keyword argument 'color'", возникавшая при сборке.
3. Интерфейс: 
   - Добавлена KPI-панель для мониторинга объема базы данных.
   - Реализована полноценная вкладка "О программе" с историей изменений.
   - Улучшена читаемость таблиц (Treeview) в темной теме.
4. Портативность: Все зависимости (Pandas, CustomTkinter, SQLite) упакованы в один исполняемый файл.

ФУНКЦИОНАЛЬНЫЕ ВОЗМОЖНОСТИ:
- Ranking: Умный подбор пар на основе баллов релевантности, эффекта и доказательности.
- Audit: Проверка качества и статуса зрелости каждой записи.
- Consensus: Поиск наиболее статистически значимых комбинаций.
- Input: Форма быстрого ввода новых данных.
- Migration: Автоматический перенос данных из версий v6, v7, v8.

ИНСТРУКЦИЯ ПО ЗАПУСКУ:
Запустите файл PhageATB_Native_v9.1.exe. База данных будет создана/использована в той же директории.
