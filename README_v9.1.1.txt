Phage + Antibiotic Selector (PhageATB) v9.1.1
============================================

Текущая версия: 9.1.1
Дата обновления: 2026-04-23

ОПИСАНИЕ:
Профессиональное программное обеспечение для анализа синергии бактериофагов и антибиотиков. 
Версия 9.1.1 включает исправления для работы с репозиторием и обновления метаданных.

ОСНОВНЫЕ ИЗМЕНЕНИЯ В 9.1.1:
1. Git & Repo: Настроен .gitignore и очищен индекс от временных файлов сборки.
2. Стабильность: Исправлены проблемы с авторизацией при пуше в удаленный репозиторий.
3. Метаданные: Обновлена информация о версии во всех внутренних модулях.

ОСНОВНЫЕ ИЗМЕНЕНИЯ В 9.1 (Нативная):
1. Архитектура: Полный отказ от Streamlit. Работа на базе CustomTkinter.
2. Интерфейс: Добавлена KPI-панель и вкладка "О программе".
3. Портативность: Единый EXE файл.

ИНСТРУКЦИЯ ПО ЗАПУСКУ:
Запустите файл PhageATB_Native_v9.1.1.exe. База данных будет создана/использована в той же директории.
